# SQL Engineer Path

An interactive study companion for DataCamp's **Associate Data Engineer in SQL** career track.

## What it includes

- A nine-course roadmap aligned to the track sequence
- Local progress tracking seeded from the supplied screenshots
- Proof-of-skill checkpoints for every course
- Three original SQL query micro-challenges
- An eight-week study sprint, persistent notes, and a knowledge check
- Responsive, dependency-free HTML/CSS/JavaScript

Open `index.html` directly, or use the GitHub Pages deployment.

> This is an independent learning companion. Course names and links belong to DataCamp; the practice prompts and interface are original.
